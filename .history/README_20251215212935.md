# Portfolio de CodeByEvans

Este portfolio está construido con **React + TypeScript + Vite** y destaca por su interactividad y diseño moderno. Incluye animaciones 3D en la sección de proyectos, transiciones suaves con **Framer Motion**, y componentes reutilizables siguiendo el sistema **shadcn/ui**.

## Características del portfolio

- **Animaciones 3D**: los proyectos se muestran en un carrusel tridimensional interactivo usando Three.js y React Three Fiber.
- **Transiciones suaves**: Framer Motion se usa para scroll, modales y microinteracciones.
- **Componentes reutilizables**: diseño consistente gracias a la librería Shadcn/UI.
- **Responsive y moderno**: adaptado a móviles y pantallas grandes, con modales animados y efectos de hover en tarjetas de proyectos.

> Los proyectos en sí se encuentran en el código del portfolio, aquí solo se describe la arquitectura y funcionalidades principales.
